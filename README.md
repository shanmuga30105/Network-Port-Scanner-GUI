# Network-Port-Scanner-GUI
A **Network Port Scanner GUI** is a graphical application that allows users to scan a target device or network to identify open, closed, or filtered ports in an easy, visual way. Unlike command-line tools, it provides buttons, input fields, and real-time results, making it user-friendly for beginners. 
A lightweight TCP port scanner with a graphical user interface built with Python and Tkinter.
Features

Simple and intuitive graphical interface

Scan single or multiple ports

Displays port status (Open, Closed, Filtered)

Fast and efficient scanning

Useful for network troubleshooting and security analysis

Technologies Used

Python (or your language)

GUI Framework (Tkinter / PyQt / etc.)

Socket Programming

How It Works

The application sends requests to specified ports on a target IP address and analyzes the response to determine whether the ports are open or closed.

Installation

Clone the repository

Install required dependencies

Run the application

Usage

Enter the target IP address

Specify port range

Click on the "Scan" button

View results in the interface

Use Cases

Network security testing

Identifying active services

Troubleshooting connectivity issues
