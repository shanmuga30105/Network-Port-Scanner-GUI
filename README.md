🔍 Network Port Scanner GUI 

A simple and user-friendly Network Port Scanner with a Graphical User Interface (GUI) built using Python. This tool allows users to scan open ports on a target system and analyze network security easily without using command-line tools.


📌 Features :
🌐 Scan ports of any IP address or domain
🔢 Custom port range selection (e.g., 1–1024)
⚡ Fast scanning using multithreading
📊 Displays open and closed ports clearly
🖥️ Interactive GUI using Tkinter
📈 Real-time scan progress

🛠️ Technologies Used
* Python 3
*Tkinter (GUI development)
*Socket Programming
*Threading
